1. To install copy game files into Hacklace_main folder.

2. Add game to "HL_AppRegistry.h" by including the game files and adding the apps 
to the list of all apps. Take "HL_AppRegistry.h" in games folder as an example.

3. Recompile the Hacklace project using Arduino and flash it to your Hacklace2.

4. Put the Hacklace into download mode and download an appropriate configuration file. 
"Hacklace_Games_Config.txt" may serve as an example.

Note: The Hacklace flash memory is limited to 32 kbyte. It can not hold all animations + all application apps + all games. Choose the ones you like.
